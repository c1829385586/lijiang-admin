# 丽江本地生活 - 管理后台

基于 Vue3 + Element Plus + Supabase 构建的丽江本地生活小程序管理后台。

## 功能模块

- 📊 **数据概览** - 展示总商家数、订单数、用户数、今日营收等核心数据
- 🏪 **商家管理** - 审核、启用/禁用商家账号
- 📋 **订单管理** - 查看和处理所有订单（酒店、美食、游玩、特产）
- 🏨 **店铺管理** - 店铺上架/下架管理
- 🎁 **商品管理** - 特产商品上下架管理
- 🖼️ **轮播图管理** - 首页轮播图添加、编辑、删除
- ⭐ **评价管理** - 查看、隐藏、删除用户评价
- 👤 **用户管理** - 查看注册用户信息
- ⚙️ **系统设置** - 修改管理员密码

## 技术栈

- Vue 3 (CDN)
- Element Plus UI
- Supabase (后端数据库)
